The Qiacuity dPCR adapter is intended to convert text file (.csv) exports from Qiagen Digital PCR Software to the dPCR Allotrope Simple Model (ASM). Qiacuity dPCR systems are digital PCR (dPCR) instruments that are used to quantify specific DNA oligonucleotides within samples that have been ‘partitioned’ into a large number of micro-reactions.

The adapter is expected to:

  • Watch a designated directory for .csv files matching the pattern of Absolute Q exports
  • Convert the exported .txt file to an Allotrope .json file
  • Return the original .csv file and the converted ASM .json file to a designated S3 bucket

The adapter is expected to produce the following output based on the provided input:

  • Input: miCURY mir-144-5p-analysis
