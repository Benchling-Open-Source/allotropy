Empower Liquid Chromatography Adapter Requirements

Revision # 1

Date: Nov 15, 2024

Background Section

  • This adapter is intended to convert exports from instrument software Empower and generate a .json file corresponding to the liquid-chromatography Allotrope Simple Model
  • Empower software was created by Waters Corporation and can automate chromatography experiment planning, executing and data analysis. In addition to working with HPLC systems, the software can be used for other chromatography applications, like LC-MS.
  • The adapter is expected to recognize the detection modality used to generate the data and generate the liquid chromatography ASM .json file using the appropriate measurement schema.

Device Control Aggregate Document / Device Control Document

The device control document within the ASM is intended to record information about device settings under which each measurement was performed. The measurement document will always contain a device control aggregate document which in turn contains device control document. The device control document is an array of objects, each recording details about a component of the instrument.

Because the same device settings are typically used for the measurement of all samples within the dataset, it is expected that these values will be repeated across multiple measurement documents with the ASM file.

Calculated Data Aggregate / Calculated Data Document

Within the ASM, the calculated data document is used to record results that are derived from calculations performed using data across different measurement documents and/or other calculated data documents . A calculated data document differs from a processed data document in that a processed data document is a facet of a measurement document and contains results derived from transformation of the data within that measurement document.

While a calculated data document records calculated results, the adapter is not expected to perform any calculations, and rather parse calculated values which appear within the instrument software file export.

All calculated data documents should be placed within the calculated data aggregate document within the liquid chromatography aggregate document, regardless of whether the calculated data is derived from measurement documents from within the same liquid chromatography document, or from measurement documents from different liquid chromatography documents.
