Generation of the ASM file

The Allotrope cell counting ASM for [instrument] data has the following structure:

Cell Counting Aggregate Document 1 total

⮑ Device System Document 1 total

⮑ Data System Document 1 total

⮑ Cell Counting Document 1 per sample/well

⮑ Measurement Aggregate Document 1 per sample/well

⮑ Measurement Document(s) 1 per sample/well*

⮑ Sample Document 1 per sample/well

⮑ Device Control Aggregate Document 1 per sample/well

⮑ Device Control Document(s) 1 per sample/well

⮑ Processed Data Aggregate Document 1 per sample/well

⮑ Processed Data Document 1 per sample/well*

⮑ Data Processing Document 1 per sample/well

⮑ Image Aggregate Document 1 per sample/well

        ⮑ Image Document 1 per sample/well*

*Note: These define the standard case for model implementation, additional use cases can occur where there will be multiple

Cell Counting Document

The cell counting document is an example of a technique document in the context of the Allotrope Simple Model, and is meant to represent each time the cell counting technique is performed. In this framework the cell counting technique is performed for each sample in the instrument software export. This means that information from each sample should be recorded in a separate cell counting document.

Device Control Aggregate Document / Device Control Document

The device control document within the ASM is intended to record information about device settings under which each measurement was performed. The measurement document will always contain a device control aggregate document which in turn contains device control document. The device control document is an array of objects, each recording details about a component of the instrument.

Because the same device settings are typically used for the measurement of all samples within the dataset, it is expected that these values will be repeated across multiple measurement documents with the ASM file.

Processed Data Aggregate Document / Processed Data Document

The processed data document contains records of information derived from the measurement data itself. As cell counting instruments are typically imaging-based instruments which rely on the processing of images to determine properties such as “cell count” or “viability”, these values are modeled as processed data. The processed data document is captured within a processed data aggregate document, to allow for recording of multiple processed data documents.
